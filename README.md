# toBuyList

## 環境開発の手順
## yarn インストール
```npm install --global yarn```
## 諸々使うものをインストール
```
yarn
```
## アプリケーションの起動
```yarn start```

## ルール(あれば追加)
* Componentsは .jsx で作る
* フロントに見えない処理(actionsとかreducersとか)は .js で作る
