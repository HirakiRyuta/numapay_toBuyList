# toBuyList

## 環境開発の手順
## yarn インストール
```npm install --global yarn```
## 諸々使うものをインストール
```
* yarn add redux
* yarn add react-redux
* yarn add redux-thunk
* yarn add redux-form
```
## アプリケーションの起動
```yarn start```

## ルール(あれば追加)
* ";" はファイルを重くするため記述しない
* Componentsは .jsx で作る
* フロントに見えない処理(actionsとかreducersとか)は .js で作る
